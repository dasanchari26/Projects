# ClothesVille – PHP E-Commerce Prototype
A simple full-stack e-commerce website built using PHP, MySQL, HTML, CSS, and JavaScript.  
This project was developed as part of a university coursework assignment.

---

## 🧰 Tech Stack

- PHP (server-side logic)
- MySQL (database)
- HTML / CSS (frontend)
- JavaScript (form validation)
- XAMPP (local development environment)

---

## 📌 Features

- User registration and login system
- Session-based authentication
- Product browsing with filters (size, category, search)
- Product detail page
- Shopping cart (session-based)
- Checkout system
- Order confirmation + order history view
- User profile view and edit functionality

## ⚙️ Setup Instructions

1. Install XAMPP (Apache + MySQL)
2. Place project folder inside: C:\xampp\htdocs\
3. Start Apache + MySQL in XAMPP
4. Import database:
	- Open phpMyAdmin
	- Create database: `f32ee`
	- Import `schema.sql`
	- Import `seed.sql`
5. Run project: http://localhost/Clothesville/index.php

## 🗄 Database Notes

- Database name: `f32ee`
- Tables:
- users
- clothesville_products
- clothesville_orders
